# Human Body Analyzer

## Installing

- Getting application

```
git clone https://github.com/im-javad/human-body-analyzer.git
```

- Go to the next-app folder

```
cd human-body-analyzer
```

- Install dependencies

```
npm install
```

- Enter the following command for run

```
npm run dev
```

- [Open in browser](http://localhost:3000/)

## Techs Used

- **Next.js**
- **TypeScript**
- **Tailwind CSS**
